# Contributing

Thanks for considering contributing. This project is open source and welcomes contributions of all sizes.

## Quick Start

1. Fork the repository and create your branch from `main`.
2. Install dependencies:

```bash
npm install
```

3. Build the project:

```bash
npm run build
```

4. Make your changes and keep the scope focused.
5. Run a quick sanity check:

```bash
npm run build
```

6. Open a pull request with a clear description of the change and why it is needed.

## Code Style

- TypeScript strict mode is enabled.
- Prefer small, focused changes.
- Keep examples and docs in sync with behavior.

## Reporting Issues

When opening an issue, include:

- What you were trying to do
- Expected behavior
- Actual behavior
- Relevant logs or error messages
- Steps to reproduce

## Security

If you discover a security issue, please do not open a public issue. Contact the maintainers directly.
